# Graph Reference #

## Red Nodes (Malicious) ##
Non-ascii characters were found or the following elements are referenced or contained with the Obj:

> /Encrypt

> /AA

> /OpenAction

> /RichMedia

> /Launch

> /JS

> /JavaScript

> /EmbeddedFile


## Yellow Nodes (Suspicious) ##
The following elements are referenced or contained with the Obj:

> Contains stream

> /JBIG2Decode

> /ObjStm

> /XFA

## White Nodes (Interesting) ##
> Object Referenced by a node but not found in PDF

## Green Nodes (Normal) ##
> Nothing malicious or suspicious detected
